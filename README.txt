Venkatesh Kuppala — Portfolio Package
======================================

This folder contains three versions of the same portfolio, plus the profile photo they all use.

FILES
-----
1. index.html
   Dark terminal / dashboard theme. Typing hero animation, skills shown as a
   database schema, experience as a log, projects as query output.
   Best as the "click through for more" version — technical, memorable.

2. index-simple.html
   Minimal, no animation, everything visible on load. No external fonts,
   loads instantly. Best as the primary link on your resume/LinkedIn for
   fast recruiter scanning.

3. index-colorful.html
   Vibrant gradient theme — color-coded stat cards, experience, and
   skill groups. Good for creative-facing roles or a more energetic first
   impression.

4. venkatesh-photo.jpg
   Your profile photo. Required by all three HTML files — keep it in the
   SAME folder as whichever index file you use.

HOW TO USE
----------
- Open any index*.html file directly in a browser to preview it.
- To publish for free: create a GitHub repository, upload these files,
  then enable GitHub Pages (Settings → Pages → deploy from main branch).
  Rename whichever version you want live to "index.html" as the entry point.
- Alternatively, drag-and-drop the folder into Netlify or Vercel for a
  free hosted link.

Update your contact info, project details, or links by editing the text
directly inside each HTML file — all content is plain text near the top
of the <body> section, no build tools required.
